Fake News Detector
📌 Description

The Fake News Detector is a web-based application that helps users identify whether a news article is real or fake. It analyzes the given text using keyword-based logic and provides an instant result in the browser.

🎯 Objective

The main objective of this project is to reduce the spread of misinformation and promote awareness about verifying news content. It supports United Nations SDG 16 (Peace, Justice and Strong Institutions).

🚀 Features
🧠 Detects fake and real news
⚡ Instant result
🎨 Simple and attractive user interface
🌐 Runs directly in browser (no backend needed)

🧠 How It Works

The system checks the input news text and compares it with predefined keywords related to fake and real news. Based on the number of matches, it predicts whether the news is real or fake.
